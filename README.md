#Python Crash Course Review Project
##-Trading Journal Manager-
###Goal
Build one complete Python application that reinforces the majority of concepts learned in Python Crash Course.
The project focuses on applying Python fundamentals, object-oriented programming, file handling, JSON storage, data analysis and visualization in one practical application.

##Features
    * Object-Oriented Programming 
    * File Handling 
    * JSON Database 
    * Exception Handling 
    * Data Analysis 
    * Matplotlib Charts 
    * Multiple Python Modules 
    * Clean Project Structure 

##Technologies
    * Python 
    * JSON 
    * pathlib 
    * Matplotlib 
    * Plotly
    * requests


### Short description:
- **main.py** – application entry point, menu, user interaction and module coordination.
- **trade.py** – defines the Trade class and stores information about a single trade.
- **journal.py** – manages the collection of trades.
- **storage.py** – saves and loads journal data from JSON files.
- **statistics.py** – calculates trade statistics.
- **api.py** – retrieves external market data.
- **plot_matplotlib.py** – generates static charts.
- **plot_plotly.py** – generates interactive charts.
- **tests.py** – contains tests for the application's core functionality.
